# Available problems for collaboration

These are opportunities for a full-fledged proof to be built.

### Problem 1: Full recompute cost

Given $n$ points and VR filtration up to dimension $q$, combinatorial growth in densifying spaces is expensive.

### Problem 2: Dynamic support update bounds

Given a point cloud trajectory $X_t$, its support graph $E_t$, a potential landmark set $L_t$, and a drift measure $\delta_t$, come up with a characterization of when support refresh preserves detection-time behaviors.


### Problem 3: Fidelity targeting

PH diagrams do not need to be equal across the board, but the tradeoff needs to be explored in a theoretical way. Experimental summaries are nice, but if this drives PH approximation theory in any way, that would be nice.
